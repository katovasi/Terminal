git_status - какой файл отслеживается гитом
:x :wq - выход из vim с сохранением
-am - сокращенный add к файлам которые ранее коммитили
ctrl C - для сохранения изменений в файде после изменения по cat
cat > - перезаписать файл
cat >> - дописать изменения в файл
git fetch - это git pull без скачивания изменений на внешнем репозитории (команда говорит о наличии разночтений внешнего и локального репозитория)
git pull -  синхронизация внешнего и локального репозитория
history - история команд
history >> command.txt - сохранить историю в блокнот
grep string *	- Поиск во всех файлах в текущем каталоге
grep string dir	- Поиск во всех файлах в каталоге dir
grep -r string .	- Рекурсивный поиск во всех файлах во всех подкаталогах
grep -r string dir	- Рекурсивный поиск во всех файлах во всех подкаталогах каталога dir
grep -R string .	- То же, что и r, но следует за символическими ссылками
